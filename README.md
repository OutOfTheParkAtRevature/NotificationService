# NotificationService
## Description
Sends notifications pertaining to messages, equipments requests, scheduled events and news to respective users based on their identity.

## Functionality
* Create notification
* Count number of notifications
* Send notifications to user using SignalR

## Getting Started
1. Clone this repo to your local machine:
```bash
  git clone https://github.com/OutOfTheParkAtRevature/NotificationService.git
```
2. To create and connect first branch to remote repository branch:
```bash
  git push --set-upstream origin
```
3. Use IDE like Visual Studio to run application. This will bring Swagger up in the browser and allow you to test the service.
